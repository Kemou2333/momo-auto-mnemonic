# 助记生成规则

## 核心宗旨

用最少的字，理清单词的不同义项为什么是这些意思。找到它们的共同本质，串成一个画面或逻辑。
看的清楚最重要——用户背单词每个词只看几秒钟。

## 生成步骤（内心推理，不输出推理过程）

1. 列出这个词的所有常见义项
2. 找到能串联所有义项的共同核心——可以是一个动作画面、一个空间意象、一个词根含义、一种类比逻辑，灵活选择最直观的角度
3. 选最合适的 note_type
4. 写出助记

## 格式规则（极其重要）

- 纯文本，不要任何 markdown 语法（不要加粗、不要标题、不要列表符号）
- 换行很重要，每个义项、每层推导单独一行，方便扫读
- 用箭头 → 表示"推出"，用 / 表示"或"，用括号补充
- 短而精为宜，不水，不能超过80字（除非极其有必要）
- 不要写总结句（比如"这就是 X 的全部逻辑"、"本质就是……"这类收尾废话）
- 不要写多余的鼓励/解释性文字
- 如果一个词很简单很直观（比如 seaside = sea + side），写一句话就够，不要硬凑内容
- 如果一个词的确有两个非常值得写的角度（比如词根拆解 + 近义辨析），才分成两条独立的 note 创建，各自选对应的 note_type，否则只建立一条助记

## 内容风格规则

### 词根词缀：只在真正有帮助时追

好的词根拆解：emerge = e-（出来）+ merge（沉入）→ 从水下浮出来
坏的词根拆解：aggressive = ag-（朝向）+ gress（走）+ -ive → 硬追拉丁语，用户看了还是不知道为什么是"激进"

原则：如果拆完词根用户能一秒 get 到含义，就拆。如果拆完还要再解释一大段，不如换个角度（比如直接讲核心画面）。
拉丁语、希腊语词源不是必须的，只在它真正能帮助记忆时才写。

### 多义词：找共同画面

这是最核心的能力。范例：

survey
核心画面：从高处扫视全局
居高看地形 → 勘测
居高看民意 → 调查问卷
居高看全貌 → 综述、总览
全是同一个动作，只是对象不同。

plain
核心：没有多余的东西
没装饰 → 朴素的
没障碍 → 平原
没修饰 → 清楚明白的（plain English）
没加料 → 原味的（plain yogurt）

dampen
核心画面：弄湿，湿了就蔫了
弄湿物体 → dampen a cloth
压低热情/声音 → dampen enthusiasm

### 近义辨析：说清差异点

essential vs necessary
essential：没有就不成其为这个东西（本质层面）
necessary：逻辑上、条件上需要

historic vs historical
historic：有历史意义的（a historic moment）
historical：与历史有关的（historical research）

### 短语/固定搭配：拆字面意思

anything but
字面：除了这个，什么都有
→ 唯独这个没有
→ 根本不、绝非

hold up
核心画面：把东西固定住不让它动
不让掉 → 支撑
不让走 → 延误
不让动 → 抢劫

### 同根词群对比

sympathy / empathy / apathy
-pathy = 感受
sym-（共同）→ 一起感受 → 同情
em-（进入）→ 进入对方感受 → 共情
a-（没有）→ 没有感受 → 冷漠

### 简单词：一句话搞定

seaside
sea + side，海边。英式偏好 seaside，美式更常说 beach。

### 否定句式常驻词：说明使用场景

budge
几乎只用于否定：won't budge / can't budge
人不挪步、物不移动、立场不变，都能用

## note_type 选择规则

### 禁用类型
永远不要使用：谐音、联想、口诀

### 允许类型与优先级

按以下顺序判断，选第一个匹配的：

优先级 1：词根词缀
适用：能拆出词根（非独立词）+ 词缀，且拆完用户一秒能 get
示例词：emerge, biology, reject

优先级 2：词源
适用：有典故、历史故事、外语来源
示例词：siren, jumbo, sandwich

优先级 3：合成
适用：由两个独立完整的单词拼成
示例词：classroom, breakthrough

优先级 4：派生
适用：词干是独立单词 + 词缀变形
示例词：disheartened, legalize

优先级 5：辨析
适用：与近义词有微妙用法差异
示例词：historic/historical, essential/necessary

优先级 6：固定搭配
适用：该词的记忆要点是它的固定搭配或短语用法
示例词：anything but, hold up, depend on

优先级 7：近反义词
适用：有强对立词群或同根词群
示例词：sympathy/empathy/apathy

优先级 8：串记
适用：与几个字母相似的词放一起记更好

优先级 9：扩展
适用：补充重要释义或相关常识

优先级 10：语法
适用：有特殊语法位置或用法限制

优先级 11：其他
适用：上述都不匹配

一个词可以同时创建两条不同类型的 note（比如一条词根词缀 + 一条辨析），但每条各自独立、各自选 type，不要把两种内容塞进同一条 note 里。

## 墨墨官方 note_type 定义参考

词根词缀：解析当前单词的词根、词缀，或解析当前单词的词根并扩展相同词根的词群。词根是英文单词的"根基"部分（非独立单词），词缀是粘附在单词上的前缀或后缀。

词源：说明单词的来源，比如起源的神话故事、历史背景等。

合成：由两个或多个具有独立意思的单词组合成一个新的单词。注意区分：tenant 不是 ten+ant 合成，那是联想。

派生：通过在一个已有的独立单词上添加前缀或后缀得到的新单词。派生与词根词缀的区别：派生中拆解出的词干是已有的独立单词，词根词缀拆分之后不是独立单词。

辨析：列出易混用的近义词，并对它们进行用法上的辨析。

固定搭配：常常搭配一起出现的词组或固定搭配，如动词+介词、形容词+名词等。

近反义词：列举单词的近义词、反义词。

串记：把意思上不相关但有相似字母组合的一批单词串联在一起记忆。

扩展：扩展补充该单词的释义、相关知识等内容。

语法：说明当前单词的语法规则。

其他：若内容不符合上述类型，归为其他。

## 速度优先

每个词的助记生成应在 15 秒内完成。核心抓住了就够，不要反复打磨措辞。
